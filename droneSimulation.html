Drone Simulation with Generated Trajectory

Includes multi-agent coordination through AV2 (Blue) drone following AV1 (Red) drone

Overview

This project implements a drone simulation through ROS2 Jazzy that follows a generated trajectory of an infinity symbol. An additional drone AV2 follows the original drone AV1 for testing out multi-agent coordination.

Key Features

  • ROS2 Jazzy
  • C++
